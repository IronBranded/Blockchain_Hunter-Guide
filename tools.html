The Investigator's Toolkit

25+ tools used by ZachXBT and professional blockchain investigators — organized by function. Sourced from ZachXBT's March 2025 published toolkit. Updated March 2026.

📌 Source
ZachXBT published his personal toolkit on Telegram (March 2025) confirming no financial incentives or sponsorships. Curated list as reported by BitPinas and OSINT Industries.

🔗 Blockchain Analysis & Wallet Tracking

Freemium
TRM Labs
Blockchain Intelligence Platform
Professional-grade transaction graphs, entity labeling, risk scoring, and "Know Your VASP" database. Used by 60+ law enforcement agencies globally. Confirmed Bybit/Lazarus attribution.
Freemium
Arkham Intelligence
Multi-Chain Explorer + Intel Marketplace
Entity graphs and a bounty marketplace for crowdsourced attribution. Posted bounties for Bybit hack addresses within 24 hours — accelerating Lazarus attribution through global investigator input.
Freemium
Cielo Finance
Multi-Chain Wallet Tracker
Track wallets across EVM chains, Bitcoin, Solana, Tron, and more. Real-time alerts when tracked wallets transact. Ranked #1 on ZachXBT's personal toolkit list.
Freemium
MetaSleuth
Visual Wallet Tracing
User-friendly visual fund flow charts. Good starting point for investigators before scaling to TRM Labs. Supports multi-chain analysis.
Free
Etherscan / Solscan
EVM / Solana Block Explorers
Foundation tools for ETH and Solana investigations. Free, public, permanent records. Filter by token, time, or counterparty.
Free
Blockchair
Bitcoin Block Explorer
Multi-chain explorer with advanced filtering and clustering support for Bitcoin investigations. Supports full transaction history export.
Free
Dune Analytics
Blockchain Data Analytics
SQL-based blockchain data querying across millions of transactions. Write custom queries to detect trends, anomalies, and suspicious patterns. Public dashboards available.
Free
MetaSuites
Browser Extension
Enriches block explorer pages with entity labels, related addresses, and ENS names. Install alongside Etherscan for enhanced on-page intelligence.

🌉 Bridge & Cross-Chain Tracking

Free
Pulsy
Bridge Activity Aggregator
Aggregates bridge activity across multiple protocols. Critical for tracking chain-hopping — the most common crypto money-laundering obfuscation technique.
Free
Socketscan
EVM Bridge Tracker
Tracks cross-chain transfers on EVM-compatible bridges. Shows source chain, destination, amount, and timing for bridge transactions.
Free
Range Protocol
CCTP Bridge Explorer
Tracks Circle's Cross-Chain Transfer Protocol (CCTP) for USDC movements between chains. Important for stablecoin laundering investigations.

🔍 OSINT & Identity Intelligence

Paid
OSINT Industries
Email / Username / Phone Lookup
Searches 400+ sources for any email, username, or phone number. Returns profiles, linked accounts, photos. ZachXBT's primary identity pivot tool.
Paid
Intel X (IntelX)
Deep Web & Database Search
Searches dark web, breach databases, and indexed private data. Find leaked credentials, forum posts, and archived pages linked to an identity.
Paid
Snusbase / LeakPeek
Breached Data Search
Searches leaked databases for emails, usernames, passwords, and IP addresses. Discovering a criminal's real email from a data breach is a common case breakthrough.
Paid
Spur.us
IP Address Intelligence
Identifies VPNs, TOR exit nodes, data center IPs, and residential proxy networks. Critical for evaluating IP addresses from VASP subpoena returns.
Paid
Cavalier
Infostealer Log Search
Searches infostealer malware logs for credentials, saved passwords, and browser history. Hudson Rock used similar data to link the Bybit hack to a Lazarus-infected developer machine.
Free
Obsidian
Investigation Organization
Create link-charts, flowcharts, and structured investigation notes connecting entities, wallets, and evidence. ZachXBT's tool for organizing multi-entity investigations.

🗄️ Archiving & Evidence Preservation

Free
Wayback Machine
Web Archive (archive.org)
Archive old versions of web pages. Crucial for capturing scam websites, social profiles, or whitepapers before suspects delete them.
Free
Archive.today
Web Snapshot Tool
Instantly create permanent, timestamped web page snapshots. Archive immediately upon discovery — suspects delete evidence within hours of detecting investigation.
Free
Impersonator
dApp Security Testing
Chrome extension for safely spoofing wallet logins to dApps. Test DeFi application security without exposing real wallets to verify legitimacy.
