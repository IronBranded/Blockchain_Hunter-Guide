Blockchain Foundations

You cannot investigate what you don't understand. This module builds foundational knowledge of how blockchains work and why they are investigable — despite the perception of anonymity.

⛓️ What is a Blockchain?

A blockchain is a publicly available, immutable ledger of every transaction ever made on that network. Think of it as a global spreadsheet that anyone can read, but from which no one can delete rows.

When Person A sends Bitcoin to Person B, that transaction is cryptographically verified by thousands of computers worldwide and permanently written into a "block" — chained to every previous block. This transparency is what makes blockchain investigation possible. Unlike traditional financial crime where investigators need warrants to access bank records, on-chain data is publicly visible to everyone.

How Transactions Work

STEP 01
Wallet Creation
A user creates a crypto wallet, generating a private key (a secret password) and a public address (like a bank account number). The address is pseudonymous — it doesn't contain a name, but it is permanently traceable.
STEP 02
Transaction Broadcast
When sending crypto, the sender broadcasts a signed transaction to the network: "I'm sending 0.5 BTC from address 1ABC… to address 3XYZ…." This is signed with the private key, proving ownership of funds without revealing the key itself.
STEP 03
Network Verification
Network nodes verify the transaction is valid — the sender has sufficient funds and the signature is cryptographically correct. Valid transactions enter the mempool (pending queue).
STEP 04
Block Confirmation
Miners or validators bundle pending transactions into a block and add it to the chain. Bitcoin: ~10 min. Ethereum: ~12 sec. Once confirmed, the transaction is permanent and irreversible.
STEP 05
Public Permanent Record
The transaction is now visible to anyone using a block explorer: sender address, recipient address, amount, timestamp, and transaction hash (TXID). This is the investigator's primary evidence source.

⚠️ Privacy Coins
Monero (XMR) and Zcash (ZEC) use cryptographic techniques to obscure sender, recipient, and amount. Significantly harder to trace but not impossible — exchanges that onboard/offboard these assets create investigable chokepoints.
🔎 Why Blockchain IS Traceable — The Investigator's Advantage

Criminals often believe crypto is "anonymous." This misconception is their vulnerability and the investigator's advantage:

  • Every transaction is public — unlike cash, there is a permanent, immutable audit trail for every movement
  • Addresses cluster together — one person controls multiple addresses that exhibit identifiable behavioral patterns
  • Exchanges require KYC — to convert crypto to fiat, criminals must eventually pass through identity verification
  • Metadata leaks identity — IP addresses, device IDs, timing patterns, and transaction errors expose real-world presence
  • Criminals brag — social media posts, luxury spending, and forum activity create OSINT leads that link pseudonyms to real people
